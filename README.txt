# ScaleImpTcl: Scale calculator tool re-implemented in Tcl
This README is a work in progress.

## Installation

### From Source
This software (is planned to use) uses tclkit to ship as a standalone binary.

To build a working tclkit:

Install metakit from https://git.jeelabs.org/jcw/metakit

Install tclvfs from https://wiki.tcl-lang.org/page/tclvfs 

For windows: Install DDE from https://wiki.tcl-lang.org/page/dde
